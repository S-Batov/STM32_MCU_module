Detailed Description

Structure containing configuration data for PWM pin alternate functions.

While assigning addresses to gate drivers PWM pins are used as digital outputs. For that process they have to be reconfigured to digital outputs and back to timer output pins upon completion of the assignment. Data needed to reconfigure the pins back to timer output pins is stored here.
